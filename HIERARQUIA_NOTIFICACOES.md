# Hierarquia das notificações

- Somente usuários com função `service` recebem notificações de vales.
- Todos os usuários de serviço ativos vinculados à mesma sessão recebem os avisos dos vales dessa sessão.
- O usuário de sessão não recebe notificações dos usuários de serviço.
- Usuários e aparelhos de outras sessões nunca recebem esses avisos.
- Cada aparelho precisa ativar as notificações uma vez.
